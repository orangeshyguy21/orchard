OpenID Discovery URL
The OpenID Connect discovery endpoint that issues clear auth tokens (CATs). Wallets fetch it to learn the provider's authorization, token, and JWKS endpoints, so it must be reachable from any wallet connecting to the mint.
Client ID
The OAuth 2.0 client identifier wallets must present when authenticating with the OIDC provider. The client should be registered as public (no client secret) with PKCE enabled and the wallet redirect URLs allowlisted.
